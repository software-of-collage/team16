<div>
<h2>Web Services</h2>

<p>Web Services 可以将应用程序转换为网络应用程序。</p>

<p>Web Services 可以被其他应用程序利用。</p>

<p>基本的 Web Services 平台是 XML+HTTP。</p>

<p>在本教程中，我们把一个 ASP.NET 程序转换为了 Web Services。<a href="/webservices/index.asp" title="Web Services教程">开始学习 Web Services</a> ！</p>
</div>


<div>
<h2>WSDL</h2>

<p>WSDL（网络服务描述语言，Web Services Description Language）是一门基于 XML 的语言，用于描述 Web Services 以及如何对它们进行访问。</p>

<p><a href="/wsdl/index.asp" title="WSDL 教程">开始学习 WSDL</a> ！</p>
</div>


<div>
<h2>SOAP</h2>

<p>SOAP 是一种简单的基于 XML 的协议，它使应用程序通过 HTTP 来交换信息。</p>

<p>在我们的 SOAP 教程中，你将了解到什么是 SOAP，以及它如何在应用程序之间交换信息。</p>

<p><a href="/soap/index.asp" title="SOAP 教程">开始学习 SOAP</a> ！</p>
</div>


<div>
<h2>RSS</h2>

<p>通过使用 RSS，您可以有选择地浏览您感兴趣的以及与您的工作相关的新闻。</p>

<p>通过使用 RSS，您可以把需要的信息从不需要的信息（兜售信息，垃圾邮件等）中分离出来。</p>

<p>通过使用 RSS，您可以创建自己的新闻频道，并将之发布到因特网。</p>

<p><a href="/rss/index.asp" title="RSS 教程">开始学习 RSS</a> ！</p>
</div>


<div>
<h2>RDF</h2>

<p>资源描述框架 (RDF) 是描述网络中资源的 W3C 标准。</p>

<p>RDF 是一个框架，用来描述网络资源，诸如网页的标题、作者、修改日期、内容以及版权信息等。</p>

<p><a href="/rdf/index.asp" title="RDF 教程">开始学习 RDF</a> ！</p>
</div>
